Funnel Diagnostic — where does each query get stuck?

Every query is assigned to the funnel stage where its journey effectively stops, measured against the baseline funnel implied by Amazon's weights (~2.7% CTR · ~14.5% click→cart · ~36% cart→purchase). Click a stage to see which queries live there — sized by True Score, with the broken ratio flagged.

← Click a funnel stage to see its queries

Baseline rates are inferred from Amazon's True Score weights (1 / 37 / 256 / 703) — a consistent yardstick, not an official Amazon benchmark. A query passes a stage when its rate reaches the baseline.

1. Rank vs. Magnitude — the reveal

Search Query Score is a rank (1, 2, 3…). This chart shows what each rank is actually worth in True Score points. The gap between rank 1 and rank 20 is almost always huge — and never obvious from the rank alone.

2. Concentration Curve — where your funnel value lives

Cumulative share of total True Score across all queries. The gold and lime markers show how few queries make up 50% and 80% of your entire search funnel value. This is your defence list.

3. Funnel Composition — points earned by stage

Every True Score bar broken into its four funnel components (Impressions × 1, Clicks × 37, Cart Adds × 256, Purchases × 703). Instantly reveals which queries are purchase-driven vs. impression-inflated.
